Directory Stucture :
-------------------

    ez-backup-esxi/
      ├── app.js
      ├── Dockerfile
      ├── docker-compose.yml
      └── package.json

Build and run docker container :
---------------
    docker-compose up --build

Enjoy it :
---------------
Your project will be built and run in a Docker container on port 3000. You can access the application at http://<Your_IP>:3000
